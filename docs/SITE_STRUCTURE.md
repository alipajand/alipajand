# Site structure

- **`/` (home)** — Best-hits scan: hero, proof, why hire, selected work, testimonials, writing teaser, hiring summary, contact. Optimized for a quick read.
- **`/portfolio`** — Full depth: about, experience, case studies, innovation, skills, education. Same content lives here when it would repeat or bury the homepage.

Use home for first impressions and conversion; use the portfolio page for evidence and narrative length.
